三位一体招生信息爬虫
====

**三位一体招生信息爬虫**用于爬取[中国教育在线](http://www.eol.cn/html/g/zjswyt/)中的浙江高校三位一体报考简章中的信息，并写入Excel
表，以供学生及生涯规划咨询师查阅。
